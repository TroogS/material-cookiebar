# Material Design Cookie Bar

## Usage
```css
MaterialCookieBar.build({
    header: "Privacy",
    message: "This site makes use of Cookies. Please refer <a href=\"/Privacy\">Privacy Policy</a> for more details.",
    buttonText: "OK"
});
```
